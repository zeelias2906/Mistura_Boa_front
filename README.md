# MisturaBoa 🍽️

Projeto front-end desenvolvido em Angular para o aplicativo de delivery do restaurante Mistura Boa.

---

## ✅ Tecnologias Utilizadas

- Angular CLI 18.2.1
- TypeScript
- SCSS
- Consumo de API com HTTPClient
- Hospedagem: [Netlify](https://www.netlify.com/)
